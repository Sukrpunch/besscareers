### 2. HMA (Hazard Mitigation Analysis)

**What is HMA?**
Detailed analysis of every possible failure mode + mitigation strategy:

```
Fire Hazard:
  ├─ Thermal runaway detection system (response < 60 sec)
  ├─ Fire suppression (foam/inert gas) 
  ├─ Isolation protocols (prevent spread)
  └─ Evacuation plan (safe egress)

Electrical Hazard:
  ├─ Arc flash protection (PPE, boundaries)
  ├─ Shock hazard mitigation (lockout/tagout)
  ├─ Grounding verification (resistance < 25 ohms)
  └─ Fault detection (protection coordination)

Environmental Hazard:
  ├─ Spill containment (secondary containment for coolant)
  ├─ Noise mitigation (sound barriers)
  ├─ Visual impact (landscaping, setbacks)
  └─ Water discharge (compliance with EPA/state)
```

**The Specialist's Job:**
Write the HMA that satisfies regulators, insurers, AND the community.

**Market Value:** +$25K–$35K
