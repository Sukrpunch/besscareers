## What Is NFPA 855?

NFPA 855 is the **Standard on the Installation of Stationary Battery Systems for Electric Vehicle Charging Stations, Stationary Energy Storage Systems, and Other Auxiliary Power Applications**. Despite the verbose title, it's the unified standard that tells you:

1. How to design a battery system for safety
2. How to install it safely
3. How to operate it safely
4. What to do if something goes wrong

### The Three Editions

## The NFPA 855 Compliance Framework: What Gets Audited

### 1. System Design & Safety Analysis

**What Gets Reviewed:**
- Hazard analysis (FMEA, risk matrices)
- Thermal runaway protection (detection + suppression)
- Fire separation distances
- Ventilation and cooling adequacy
- Electrical safety (arc flash analysis, grounding)

**Compliance Checklist:**
```
☐ Hazard analysis completed and documented
☐ Thermal runaway detection system specified
☐ Suppression agents selected (Class C foam, dry chemical, inert gas)
☐ Separation distances from occupied buildings confirmed
☐ Ventilation analysis shows safe gas clearance
☐ Electrical design meets NEC Article 706 requirements
☐ UL 9540 certificate obtained for battery system
```

### 2. Installation & Construction

**What Gets Verified:**
- Proper cable routing and protection
- Grounding and bonding completed
- Transformer and switchgear installation per code
- Fire detection systems operational
- Emergency shutdown systems functional

**Field Audit (Site Visit):**
```
Phase 1: Foundation (During construction)
  ☐ Concrete strength verified (28-day testing)
  ☐ Cable trays and conduit per plans
  ☐ Grounding resistance < 25 ohms (confirm with clamp meter)

Phase 2: Equipment Installation (Before energization)
  ☐ Inverter mounting bolts torque-verified
  ☐ Battery rack leveling confirmed (±1mm tolerance)
  ☐ Fire detection system acceptance tested
  ☐ Emergency shutdown accessible and functional

Phase 3: Electrical Verification (Before grid connection)
  ☐ Insulation resistance testing (1000V megaohm test)
  ☐ Continuity of grounding conductors
  ☐ Transformer secondary winding polarity confirmed
```

### 3. Commissioning & Testing

**System Acceptance Testing (SAT) Per NFPA 855:**
- **Thermal Runaway Simulation** (if lithium): How quickly does the detection system respond? Does suppression trigger automatically?
- **Emergency Shutdown Test** (full cycle): E-stop button → full system shutdown in < 1 second
- **Arc Flash Test** (where applicable): Verify protection systems can clear a fault in specified time
- **Frequency Response** (grid support): Verify inverter response time and accuracy per FERC Order 2222

**Pass/Fail Criteria (No Exceptions):**
```
Thermal Runaway Response:    < 60 seconds from detection to suppression
Emergency Shutdown:          < 1 second (hard stop)
Arc Flash Clearing Time:     < 100ms for high-fault current locations
Frequency Response:          ±2% accuracy, 500ms response time
```

### 4. Operations & Maintenance

**What Operators Must Do (Ongoing):**
- Monthly visual inspections (check for coolant leaks, visible damage)
- Annual performance testing (run through all safety interlocks)
- 5-Year Major Maintenance (replace coolant, inspect wiring, test backup systems)
- Continuous Monitoring (SCADA + alarms for thermal/electrical anomalies)

**O&M Documentation:**
```
☐ Operations & Maintenance Manual (specific to this installation)
☐ Training certificates for all operators (NFPA 855 training + site-specific)
☐ Monthly inspection log (signed by trained personnel)
☐ Annual test reports (thermal runaway, E-stop, frequency response)
☐ Incident/anomaly log (every fault or near-miss documented)
```
