# BESScareers

Battery Energy Storage career guides and salary benchmarks.

**Live:** https://besscareers.com

## Getting Started

```bash
npm install
npm run dev
# Open http://localhost:3000
```

## Deploy to Vercel

1. Push to GitHub
2. Connect to Vercel (automatic)
3. Done!

## About

Part of the BESSjobs ecosystem. Built with Next.js 14, Tailwind CSS, and TypeScript.
